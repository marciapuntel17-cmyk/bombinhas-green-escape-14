== Description ==

Tema WordPress elegante e responsivo para pousadas e hotéis, com design inspirado na Mata Atlântica e no mar de Bombinhas. Oferece um layout one-page completo com seções para hero, sobre, acomodações, experiências, localização e contato.

== Recursos ==

* Design responsivo e moderno
* Layout one-page otimizado
* Customizer completo para personalização
* Cores e gradientes inspirados na natureza
* Botão WhatsApp flutuante
* Formulário de reserva integrado
* SEO otimizado
* Schema.org para negócios locais
* Google Fonts integrado

== Instalação ==

1. Faça upload dos arquivos para /wp-content/themes/pousada-bombinhas/
2. Ative o tema no painel do WordPress
3. Vá em Personalizar para configurar o tema
4. Adicione suas imagens nas seções correspondentes
5. Configure as informações de contato

== Customização ==

Acesse Aparência > Personalizar para:

* Personalizar textos de todas as seções
* Adicionar suas próprias imagens
* Configurar informações de contato
* Ajustar endereço e dados da pousada
